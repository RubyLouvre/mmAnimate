avalon.fn.delay(duration)

描述：

delay会将其列队中没有执行的动画都向后延迟duration秒执行。

参数：

duration
必需。Number。动画执行时间，单位为ms。

返回值：

avalon实例

其实此方法相当于avalon.fn.fx(duration), 但fx方法会做许多其他操作, 而它不会。

例子 
           require("mmAnimate", function(avalon) {
                avalon.define("fx", function(vm) {
                    vm.effect1 = function() {
                        avalon(this).slideUp(300).fx(2000).fadeIn(400).hide(1000).show(1000);
                    }
                      vm.effect2 = function() {
                        avalon(this).slideUp(300).fadeIn(400).hide(1000).show(1000);
                    }
                })
                avalon.scan()
            })
