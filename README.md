mmAnimate
=========

avalon的三柱臣之一（ 路由，动画，AJAX）

它的本体只能在IE10或标准浏览器上运行(它是使用CSS3 keyframe动画实现的)，想兼容IE6-9,请先引入jQuery1.9，它会把jQuery对应的方法再包装

用法与jQuery的保持一致
