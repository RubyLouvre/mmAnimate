count

描述：

指定动画的播放次数, 要求是非负整数或infinite

返回值：

avalon实例

例子 
            require(["mmAnimate"], function(avalon) {
                avalon.define("fx", function(vm) {
                  
                    vm.effect = function() {
                       avalon(this).animate({top: 400},{count: 10, duration: 1000})
                    }

                })
                avalon.scan()
            })
