avalon.fn.show(duration, props)

描述：

用一种优雅方式显示匹配的元素。它分别对元素高宽以及所有内外边距，边框与透明度变回没有隐藏之前的值。

参数：

duration
可选。Number。动画执行时间，单位为ms，默认为500ms。
可选。Function。动画结束时的回调函数
props
可选。Object。各种配置。

返回值：

avalon实例

props配置对象可以指定各种CSS属性，如width, height, color, fontSize, opacity等进行动画变化， 还支持scrollTop与scrollLeft这两个属性进行动画变化。还可以指定动画开始前的回调函数before,动画结束时的回调函数after, 动画是否进行倒退reverse....

例子 
            require(["mmAnimate"], function(avalon) {
                avalon.define("fx", function(vm) {
                    vm.array = ["a", "b", "c", "d", "e"]
                    vm.effect = function() {
                        for (var i = 0, el; el = this.children[i++]; ) {
                             if (avalon.isHidden(el)) {
                                avalon(el).show(1500);
                            } else {
                                avalon(el).hide(1500);
                            }
                        }
                    }
                })
                avalon.scan()
            })
